// HOW IT WORKS

仕組みはシンプル

ゲームのメモリには一切触れず、公式ログファイルだけを読む安全な設計。

ChatLog*.txt
PSO2 NGS が自動生成するログファイルを監視
→
Watcher
Node.js が新着行をリアルタイム検知
→
WebSocket
localhost 経由で UI に即座に送信
→
表示 / TTS / 翻訳
フィルタリング → 表示 → 読み上げ・翻訳
// FEATURES

主な機能

チャットをもっと快適に。見やすく、聞きやすく、わかりやすく。

リアルタイム表示
ゲームが書き出す ChatLog*.txt を監視し、新着メッセージのみ即座に表示します。
REALTIME
チャンネルフィルター
PARTY / WHISPER / PUBLIC / TEAM / GUILD を個別に ON/OFF。見たいチャンネルだけ表示。
FILTER
翻訳機能
6種類の翻訳エンジンに対応。Google / MyMemory は無料・キー不要で即使用可能。
TRANSLATE
TTS 読み上げ
VOICEVOX / 棒読みちゃん / Web Speech API の 3 エンジンに対応。
TTS
スパム・ブロック
重複メッセージ・Flood 検知の自動スキップ。Regex フィルターで細かく制御可能。
FILTER
テーマ & カスタマイズ
8 種類のプリセットテーマ＋カスタムカラー。背景画像（URL / ローカル）にも対応。
THEME
ブックマーク & ブロックリスト
気になったメッセージを保存。特定プレイヤーのメッセージを永続的に非表示。
MANAGE
自動アップデート
起動時 + 30 分ごとに GitHub Releases を確認。インストールタイミングはユーザーが決定。
AUTO UPDATE
完全ローカル動作
ユーザーデータは一切収集・送信しません。すべて %APPDATA%\NYANNAPP\ にローカル保存。
PRIVACY
DLC / プラグイン
.zip ファイルからプラグインをインストール・管理。プラグイン UI をアプリ内に直接埋め込み表示。自動起動 ON/OFF も設定可能。
DLC
チャット履歴
起動時に過去 N 時間分のログを自動読み込み。NGS / Classic 両対応。次回から同じ設定を記憶するオプション付き。
HISTORY
VOICEVOX キャッシュ & 単語置換
音声をローカルキャッシュして高速再生。読み上げ前のテキスト置換ルールを自由設定（棒読みちゃん辞書内蔵）。
VOICEVOX
翻訳テキストの色分け
翻訳結果の表示色を「言語別」または「チャット（プレイヤー・チャンネル）別」に自由設定可能。設定は即時反映されます。
TRANSLATE
NYANN Browser
アプリ内ブラウザウィンドウ。外部ページをアプリのテーマに合わせて表示できます。UI は日本語 / English に対応。
BROWSER
カスタムアプリアイコン
好きな画像をアプリウィンドウのアイコンに設定可能。自分だけのオリジナル外観に。
CUSTOM
// PLUGINS

対応プラグインも多数

.zip を入れるだけで導入できる拡張プラグインを多数配布中。ここでは一例をご紹介。

ActionLog Viewer
PSO2:NGS の ActionLog（アイテム取得・トレード・N-Meseta・倉庫）をリアルタイム表示。タイトルバーから N-Meseta 履歴もドロップダウンで確認可能。
LOG
NYANN2FA
TOTP 認証コード生成プラグイン。ワンタイムパスコードをタイトルバーに直接表示し、他アプリに切り替えずにコピーできます。
SECURITY
PSO2 LOG DC
ライブチャットを Discord Webhook に自動転送。チャンネル種別（PUBLIC / PARTY / WHISPER など）ごとにフィルタし、色分け Embed で見やすく表示。
DISCORD
TCP Ping 630
PSO2 NGS サーバーへの接続状態を常時監視。TCP Ping をリアルタイム表示し、切断時は「630」エラーとしてすぐ気づけます。
NETWORK
NYANN Debug Console
アプリ本体（main process）とすべてのプラグイン（renderer）の Debug ログを一箇所にまとめてライブ表示。トラブルシューティング用。
DEBUG
NYANN Discord RPC
アプリの状態を Discord のリッチプレゼンスに反映。表示するステータス文言は自由にカスタマイズできます。
DISCORD
NYANNP Discord TTS Bot
Discord ボイスチャンネルで読み上げを行う専用 Bot（別配布・Python 製）。VOICEVOX / edge-tts / gTTS に対応し、サーバーごとに読み上げ設定を保存できます。
DISCORD BOT
Live Chat Color
チャンネルごとに名前色・吹き出し背景色を自由設定。アルファ値まで指定できるカスタムカラーピッカーと、Bubble 不透明度 / Blur スライダーも搭載。テキスト色や翻訳文の配色まで細かくカスタマイズできます。
CUSTOM
NYANN Sound Alert
チャンネルごと・プレイヤーごとにチャット通知音を再生。通知音は自由に選択・変更できます。
SOUND
SHORTCUTS
タイトルバーのクイックアクセスパネルから .exe / .lnk / .bat / .ahk / .url を素早く起動。よく使うツールをワンクリックで呼び出せます。
TOOL
NyannThemes
カスタム CSS テーマをインポートしてアプリ UI 全体（インラインスタイル含め）を丸ごと着せ替え。他プラグインのパネルやポップアップにも自動適用され、個別設定は不要。ピクセル風テーマを標準搭載。
THEME

// SAFETY

ゲームルールに違反しない

SEGA の利用規約・チートツール規制の観点から、本アプリは完全にセーフです。

  • ✓
    ゲームのメモリ・プロセスに一切アクセスしない
  • ✓
    PSO2: NGS が公式に出力するログファイル（テキストファイル）のみを読む
  • ✓
    ゲームの通信・サーバーに一切干渉しない
  • ✓
    ゲームクライアントを改変しない
  • ✓
    自動操作・マクロ機能は持たない
